# Mixed-Reality-Project
This is a Mixed reality project built usinf Meta Spatial SDK. 

The Application is Relax VR , the application drops the user in a calming enviroment while playing realxing and breathing excercise videos, the user can see a room and a video plaing and the room is open on all sides allowing user to view calming environments. The sue also has access to changing environment to choose the one they feel most relaxed in.
